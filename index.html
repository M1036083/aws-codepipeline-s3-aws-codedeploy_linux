Jenkins-AWS-CodePipeline Demo

You have successfully created a pipeline that retrieved this source application from an Amazon S3 bucket and deployed it Ubuntu Amazon EC2 instances using AWS CodeDeploy.

Steps included

Developer commits code to git->code change triggers new code pipeline execution in AWS->pull latest source code from git->trigers build in jenkins ->builds the artifacts and upload the artifacts to s3 bucket aws and returs control to aws code pipeline ->aws code pipeline now move to deployment or beta stage->it this stage it invokes aws code deploy which retrieves the package from s3 bucket and deploys to target deployment group
